# aws

AWS notes, guides, and write-ups — and the source for AWS posts on the blog.

Blog posts live under `docs/<topic>/<post>.md`. Any doc with `blog.publish: true`
in its frontmatter is picked up by the blog's sync step (see `adi-blog/sources.yml`)
and published automatically. See [`docs/_template/post-template.md`](docs/_template/post-template.md)
for the frontmatter contract.
